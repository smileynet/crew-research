## Cross-Cutting Findings

1. **Research output format fragmentation** (highest priority): three different templates for research findings exist across research-methodology (inline), research-output (canonical), and research-methodology/references/dispatch-pattern.md. Consolidate on research-output.
2. **Study-* schema drift**: study-reference and study-all-references write the same `.memory/{name}-reference.md` file with incompatible section schemas, while sharing skip-if-exists logic that assumes compatibility.
3. **`.references/` path inconsistency**: study-reference and study-all-references use bare `references/{name}` in frontmatter examples and subagent prompts, while init-project, project-audit, and project-cleanup actively enforce renaming bare `references/` to `.references/`. The study skills would generate docs pointing at a path the audit skills flag as wrong.
